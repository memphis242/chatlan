# `chatlan` - a TLS-secured home LAN chat system
Of course, there are many different chat applications out there. But I'm making
one that is very specifically what I need it to be - secure and supported on any
of the devices I'd have at home (e.g., embedded device, laptop TUI, mobile phone)
that is closed off to the rest of the world. I'll learn a lot and get something
specific to my needs out of it.

The vision for the initial chat system includes:
- the chat server (likely on a Raspberry Pi)
- TUI client application
- a Wi-Fi embedded endpoint /w an LED display of recent chats
- a local Certificate Authority
